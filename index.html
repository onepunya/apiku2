API Documentation

Welcome to the onesytex API for our futuristic service. Below you will find information about the available endpoints and how to use them, powered by advanced AI technology.

See Endpoints

Usage

To interact with the endpoints, send a GET request to the desired endpoint with the required query parameter:

  • EXAMPLE
  use
  {base_url}/api/{endpoint}?{req}={query}
  
  example 
  https://api.onesytex.my.id/api/prodia?prompt=cute%20girl

Response

The response will be in JSON format with the following structure:

  {
    "status": 200,
    "creator": "mr.one",
    "data": {
      "response": "Response data will be provided here"
    }
  }
